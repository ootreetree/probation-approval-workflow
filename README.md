# 试用期转正业务审批流程

这是一个试用期转正业务审批流程的可视化网页展示。

## 功能特点

- 📋 完整的审批流程展示（流程触发 → 直属上级审批 → HRBP审批 → 逐级审批）
- 👤 员工基础信息展示
- ✍️ 员工自评内容查看（从另一平台抓取）
- 📝 上级评价表单填写
- ⚠️ 培养任务未完成醒目提示
- 🔄 退回自评功能
- 📜 审批历史记录

## 流程说明

**触发条件**：员工入职第 135 天，由人力资源虚拟节点自动触发

**审批流程**：
1. 直属上级审批 - 填写考核结论、评价文字、评分
2. HRBP审批 - 人力资源业务合作伙伴审核
3. 逐级审批 - 上级领导逐级审批确认

## 使用方法

直接在浏览器中打开 `index.html` 文件即可查看完整的审批流程页面。

## 页面构成

- **员工基础信息** - 工号、部门、岗位、职级、入职日期等
- **员工自评内容** - 员工填写的转正自评表内容
- **上级评价内容** - 审批人填写的评价和考核结论
